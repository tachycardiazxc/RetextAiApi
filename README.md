<h3>Installation:</h3>

1) Install python 3.8
2) Open CMD
3) CMD: ```pip install virtualenv```
4) Open any folder
5) Install project here
6) Go to the project directory
7) CMD:

```
virtualenv -p=3.8 venv
.\venv\scripts\activate
pip install -r requirements.txt
python main.py
```
